3. What is event bubble ?


Answer: In event bubble, the event first try to capture the innermost element then propagated to its root element.


4. What is a callback function and why will you use it ?


Answer : Callback Function is a function which passed into another function as an argument.I will use Callback Function to make sure a function will not run before a task is completed but will run after the task has completed
